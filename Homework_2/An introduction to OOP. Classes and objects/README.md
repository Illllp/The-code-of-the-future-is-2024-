# Пример программы на Python

## Введение

В этом примере мы создаем класс `Car`, который имеет атрибуты `make` (марка автомобиля), `model` (модель автомобиля) и `year` (год выпуска). Даем ему также метод `display_info()`, который выводит информацию о машине (марка, модель и год).

## Код

```python
class Car:
    def __init__(self, make, model, year):
        self.make = make
        self.model = model
        self.year = year

    def display_info(self):
        print(f"Марка: {self.make}, Модель: {self.model}, Год выпуска: {self.year}")
```

## Обсуждение

В этом коде мы создали класс `Car`, который может хранить информацию о марке, модели и годе выпуска автомобиля. Метод `display_info` позволяет вывести эту информацию в удобном для чтения формате.

## Заключение

Это простой пример использования классов и объектов в Python. Он демонстрирует, как можно эффективно работать с объектно-ориентированным программированием.
